# Simran Bhambhani — Portfolio Website

Personal portfolio built with pure HTML, CSS, and JavaScript.

## 🚀 Run Locally
Just open `index.html` in your browser — no build step needed.
For live reload: use VS Code Live Server extension.

## 📸 Add Your Photo
Place your profile photo in the root folder named: `shimmy_profile.jpeg`

## 🌐 Deploy to GitHub Pages
1. Push this folder to a GitHub repository
2. GitHub Actions auto-deploys on every push to `main`
3. Enable GitHub Pages in repo Settings → Pages → Branch: gh-pages

## 📁 Files
- `index.html` — All page sections
- `styles.css` — All styles and CSS variables
- `script.js` — Scroll animations, counters, navbar
- `shimmy_profile.jpeg` — Your profile photo (add this)
- `.github/workflows/deploy.yml` — Auto-deploy to GitHub Pages
